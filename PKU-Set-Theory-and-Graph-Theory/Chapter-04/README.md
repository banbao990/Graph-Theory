# 第四章 平面图与图的着色

## 4.1 平面图

+ 实际应用
  + 单面印刷电路板的设计
  + 大规模集成电路的布线
  + 交通, 通讯, 城市建筑等
+ **可平面图**与**平面图**
  + 若能把图 $G$ 画在一个平面上, 使得任何两条边都不相交, 则称 $G$ **可嵌入平面**, 称 $G$ 为**可平面图**
  + **可平面图**在平面上的的一个**嵌入**(**画法**)称为**平面图**
+ 举例 
  + $K_4$ 是**可平面图**, $(b)(c)$为 $K_4$ 的两个平面嵌入, $(b)(c)$为**平面图** 

```C
只有o是结点,其他的都是边

     |  \ /  |         |    /  |        |                /  / \  \
```
